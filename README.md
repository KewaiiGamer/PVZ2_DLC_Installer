## About The Project

This project is supposed to make it easier for anyone to install PVZ DLC Edition without complicated steps. it should require as few steps as possible to minimize human intervention

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- GETTING STARTED -->
## Getting Started

This script assumes you have obtained PVZ legally. we do not support unethically obtained copies of PVZ

### Prerequisites

* Steam
* PVZ Installed on your system

### Usage

1. Simple one line script:
   ```sh
   curl -sSL https://raw.githubusercontent.com/KewaiiGamer/PVZ_DLC_Installer/refs/heads/main/pvz_dlc_installer.sh -o /tmp/pvz_dlc_installer.sh && bash /tmp/pvz_dlc_installer.sh
   ```
2. Follow the script instructions
3. Start game!
